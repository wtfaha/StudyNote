
# C#語法
*****  
NotifyIcon 類別、Timer 類別  
*****  
+ ### NotifyIcon 類別  
  指定在通知區域中建立圖示的元件。 這個類別無法被繼承。  

+ ### Timer 類別  
	+ Timer.Interval屬性  
	取得或設定引發 Elapsed 事件的間隔 (以毫秒為單位)  
	+ Timer.Enabled 屬性  
	取得或設定值，表示 Timer 是否應引發 Elapsed 事件  
	+ Timer.Elapsed 事件  
	發生於間隔耗盡時  
	+ Timer.Start 方法 ()  
	將 Enabled 設定為 true，開始引發 Elapsed 事件
