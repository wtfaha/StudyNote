+ ### 匯流排 
	![](https://cdn.discordapp.com/attachments/398435316673019905/417173091995615272/unknown.png)  
	+ 資料匯流排 :  
		雙向  
		```
		個單元間傳送資料的管道
		一次能傳輸的資料量稱為匯流排寬度
		```
	+ 位址匯流排 :  
		單向  
		```
		CPU向外傳送位址訊息的管道
		N位元的位址匯流排有N條位址線，CPU可定址的最大空間為2<sup>N<sup>Bytes
		```
	+ 控制匯流排 :  
		單向  
		```
		CPU向外傳送控制訊號的管道
		```
